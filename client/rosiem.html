Testing annyang speech recognition

Only works in Chrome

1) Click Setup to configure the service

2) Click Start to listen for "Rosie" and "Set Date". You will be prompted to enable microphone.

If the system hears either trigger, it puts up an alert. There is a long lag. It will keep listening after it recognizes a trigger. This behaviour is configurable.

The JS is so small because it is just a simple way to call webkit SpeechRecognizer
